java 编码规范

必须遵守

  1. 指定doctype 
    <!DOCTYPE html>

  2. java script 文件不要混杂在html 中，单独提出放入js文件中(如确实需要打破此规程，需说明原因)

  3. 不要写写行内css, css 需要单独放入一个文件(如确实需要打破此规程，需说明原因) 

    例如不要这样写：<div style="color:red;"></div>
